## 📊 КАК РАБОТАЕТ МАТЕМАТИКА:

### 1. Пользователь меняет валюту
```
Profile → Edit → Currency → VND → Save
localStorage.user.currency = "VND"
```

### 2. Создание записи
```
Input:
- Name: "Pho Soup"
- Price: 50,000 VND
- Quantity: 1

Backend:
totalAmount = 50,000 * 1 = 50,000 VND
usdAmount = convertToUSD(50,000, "VND")
          = 50,000 / 24,500
          = 2.04 USD ✅

Сохраняется:
Entry {
  pricePerUnit: 50,000
  currency: "VND"
  usdAmount: 2.04  ← Правильно!
}
```

### 3. Начисление поинтов
```
points = calculatePoints(2.04, "food", userId)
       = 2.04 * 1.0 * 1.0
       = 2.04 points ✅

User.points += 2.04
```

### 4. Отображение в Wallet
```
totalSavings = SUM(entries.usdAmount)
             = 2.04 USD

Frontend display (VND):
convertCurrency(2.04, "VND")
= 2.04 * 24,500
= 49,980 VND ≈ ₫50,000 ✅
```

## 🔍 ОТЛАДКА:

### Проблема: Сумма не прибавляется

**Чек-лист:**
1. ✅ Сервер перезапущен с очисткой .next?
2. ✅ Валюта VND сохранена в Profile?
3. ✅ В localStorage currency = "VND"?
4. ✅ В логах PowerShell есть "50000 VND = 2.04 USD"?
5. ✅ В Network (F12) POST /api/entries/create → 201?

**Если логов нет:**
- Запись не создалась
- Проверьте currency в Profile
- Проверьте localStorage (F12 → Console):
  ```javascript
  JSON.parse(localStorage.getItem('user'))
  // Должно быть: { currency: "VND", ... }
  ```

**Если логи есть, но Wallet не обновляется:**
- Нажмите F5 (обновить страницу)
- Или перейдите в другую вкладку и обратно
