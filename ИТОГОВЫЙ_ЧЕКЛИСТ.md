## 🚀 КОМАНДЫ ДЛЯ ЗАПУСКА:

### Вариант 1: С чистой базой (рекомендуется!)

```powershell
# Остановите сервер
Ctrl + C

# Очистите всё
Remove-Item prisma\dev.db
Remove-Item -Recurse -Force .next

# Пересоздайте
npm run db:init

npm run dev
```
