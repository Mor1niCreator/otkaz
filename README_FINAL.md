## 🔧 Если не запускается:

### Проблема с .env (BOM)
```powershell
# PowerShell 7+
pwsh -c "(Get-Content .env -Raw) | Set-Content .env -Encoding utf8NoBOM"

# Или удалите .env и создайте заново
```

### Проблема с портом 8080
```cmd
# Найдите что использует порт
netstat -ano | findstr :8080

# Убейте процесс
taskkill /PID <PID> /F
```

### Полный сброс
```cmd
docker compose down -v
docker system prune -f
docker volume prune -f
docker compose up --build
```
