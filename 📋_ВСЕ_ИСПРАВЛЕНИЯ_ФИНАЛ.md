### 2. Дублирование целей

**Файл:** `app/goals/page.tsx`

**Проблема:**
```typescript
// БЫЛО:
if (data.goals.length === 0) {
  await createDefaultGoals(userId);
  // ↓ createDefaultGoals вызывал loadGoals()
  // ↓ loadGoals снова вызывал createDefaultGoals()
  // ↓ БЕСКОНЕЧНЫЙ ЦИКЛ → дубли!
}
```

**Исправление:**
```typescript
// СТАЛО:
if (data.goals.length === 0 && !localStorage.getItem('defaultGoalsCreated')) {
  await createDefaultGoals(userId);
  localStorage.setItem('defaultGoalsCreated', 'true'); // ✅ Флаг
}

// createDefaultGoals больше НЕ вызывает loadGoals()
// Обновляет state напрямую: setGoals(createdGoals)
```

**Результат:**
- Цели создаются ОДИН раз ✅
- Нет дублей ✅
- Ровно 4 цели ✅

### 3. Сумма отказов (totalSavings)

**Файл:** `pages/api/goals/list.ts` (исправлено ранее)

**Проблема:**
```typescript
// БЫЛО:
const totalSavings = user?.points || 0; // ❌ Points ≠ Savings
```

**Исправление:**
```typescript
// СТАЛО:
const entries = await prisma.entry.findMany({
  where: { userId },
  select: { usdAmount: true },
});
const totalSavings = entries.reduce((sum, e) => sum + e.usdAmount, 0);
// ✅ Savings = реальная сумма записей
```

**Результат:**
- Savings = SUM(entries.usdAmount) ✅
- Goals показывают правильный прогресс ✅
