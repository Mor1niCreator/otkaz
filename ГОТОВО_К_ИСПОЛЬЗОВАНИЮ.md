## 🎯 Протестируйте все функции:

### 1. Добавление записи
1. Откройте **Calendar**
2. Нажмите **☕ Кофе** (или "+ Своя запись")
3. Заполните форму
4. Нажмите **Сохранить**
5. ✅ Должно появиться: **"+3.0 поинтов! 🎉"**

### 2. Проверка Wallet
1. Откройте **Wallet** (💰 Кошелёк)
2. ✅ Должна показаться статистика БЕЗ ошибок
3. ✅ График должен отобразиться
4. ✅ Points и Rank видны

### 3. Смена языка
1. Откройте **Profile** (👤 Профиль)
2. Нажмите **Изменить**
3. Смените язык на **Русский 🇷🇺**
4. Нажмите **Сохранить**
5. ✅ Страница перезагрузится
6. ✅ Весь интерфейс на русском!

### 4. Смена валюты
1. В **Profile** → **Изменить**
2. Выберите, например, **VND - Vietnamese Dong (₫)**
3. Нажмите **Сохранить**
4. ✅ Валюта сохранится

### 5. Создание цели
1. Откройте **Goals** (🎯 Цели)
2. Нажмите **+ Новая**
3. Введите название и сумму
4. ✅ Цель создастся
5. ✅ Прогресс-бар отобразится

### 6. Crypto-ROI
1. В **Goals** прокрутите вниз
2. Нажмите **🚀 Рассчитать Crypto ROI**
3. ✅ Таблица с топ-10 криптовалютами
4. ✅ Показывает рост и вашу потенциальную стоимость
