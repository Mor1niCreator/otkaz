## 🔍 ЕСЛИ БУДУТ ОШИБКИ:

### Ошибка: "table does not exist"
```powershell
Remove-Item prisma\dev.db
npm run db:init
npm run dev
```

### Ошибка: "Fast Refresh error"
```powershell
Ctrl + C
Remove-Item -Recurse -Force .next
npm run dev
```

### Ошибка: Build фейлится
```powershell
# Проверьте что frontend/ удалена:
dir frontend
# Если есть - удалите вручную:
Remove-Item -Recurse -Force frontend
```

### Полная переустановка
```powershell
Ctrl + C
Remove-Item prisma\dev.db
Remove-Item -Recurse -Force .next
Remove-Item -Recurse -Force node_modules
npm install
npm run db:init
npm run dev
```
