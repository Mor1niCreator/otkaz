# 🪟 Полная установка на Windows

## ❌ Проблема: "npm не распознано"

Это означает, что **Node.js не установлен** на вашем компьютере.

---

## ✅ Решение: Установите Node.js

### Шаг 1: Скачайте Node.js

1. Откройте браузер
2. Перейдите на **https://nodejs.org/**
3. Скачайте **LTS версию** (рекомендуется)
   - Например: "20.11.0 LTS" (зелёная кнопка)
4. Запустите установщик `node-v20.x.x-x64.msi`

### Шаг 2: Установите Node.js

1. Следуйте инструкциям установщика
2. **Важно:** Оставьте все галочки по умолчанию
3. Особенно: "Automatically install necessary tools" ✅
4. Нажмите "Install"
5. Дождитесь завершения

### Шаг 3: Перезапустите PowerShell

**ОБЯЗАТЕЛЬНО закройте и откройте PowerShell заново!**

### Шаг 4: Проверьте установку

Откройте PowerShell и выполните:

```powershell
node --version
npm --version
```

Должно показать версии (например, `v20.11.0` и `10.2.4`)

## 🚀 Теперь запускайте проект!

### Перейдите в папку проекта


## 🐛 Возможные проблемы

### Проблема 1: "npm не распознано" (даже после установки)

**Решение:**
1. Закройте PowerShell полностью
2. Откройте заново
3. Попробуйте снова

Если не помогло:
1. Откройте "Панель управления"
2. "Система" → "Дополнительные параметры"
3. "Переменные среды"
4. Проверьте, есть ли в PATH:
   - `C:\Program Files\nodejs\`

### Проблема 2: Ошибка при `npm install`

**Решение:**
```powershell
# Очистить кеш npm
npm cache clean --force

# Попробовать снова
npm install
```

### Проблема 3: "Prisma migrate failed"

**Решение:**
```powershell
# Удалить старую базу
Remove-Item prisma\dev.db -ErrorAction SilentlyContinue

# Создать заново
npm run db:init
```

### Проблема 4: Порт 3000 занят

**Решение:**
```powershell
# Использовать другой порт
npm run dev -- -p 3001
```

Откроется на http://localhost:3001

### Проблема 5: Антивирус блокирует

**Решение:**
1. Временно отключите антивирус
2. Запустите `npm install`
3. Включите антивирус обратно
