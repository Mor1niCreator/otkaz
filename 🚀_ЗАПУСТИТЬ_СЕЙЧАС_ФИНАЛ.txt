🔍 ЕСЛИ БУДУТ ОШИБКИ:

   Ошибка "table does not exist":
   Remove-Item prisma\dev.db
   npm run db:init
   npm run dev

   Ошибка "Fast Refresh":
   Ctrl + C
   Remove-Item -Recurse -Force .next
   npm run dev

   Полная переустановка:
   Remove-Item prisma\dev.db
   Remove-Item -Recurse -Force .next
   Remove-Item -Recurse -Force node_modules
   npm install
   npm run db:init
   npm run dev
