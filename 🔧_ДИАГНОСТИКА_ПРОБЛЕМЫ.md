# 🔧 ДИАГНОСТИКА ПРОБЛЕМЫ С WALLET

## ❌ ПРОБЛЕМА:

После смены валюты (USD → RUB/VND):
- ✅ Запись создается
- ✅ Поинты начисляются
- ❌ Сумма НЕ прибавляется в Wallet

---

## 🔍 ПРОВЕРИМ ВЕСЬ ПУТЬ:

### Шаг 1: Создание записи

**Выполните в PowerShell после перезапуска:**

```powershell
Ctrl + C
Remove-Item -Recurse -Force .next
npm run dev
```

### Шаг 2: Тестовый сценарий

```
1. http://localhost:3000
2. Register/Login
3. Profile → VND → Save (дождитесь reload!)
4. Calendar → + Custom:
   - Name: "Test Entry"
   - Price: 100000
   - Save
```

### Шаг 3: Проверьте логи в PowerShell

**Должно быть:**
```
Entry: Test Entry - 100000 VND = 4.08 USD
Points earned: 4.08 (from 4.08 USD)
POST /api/entries/create 201 in XXXms
```

✅ Если есть → запись создана в БД!
❌ Если нет → запись не создалась!

### Шаг 4: Проверьте Console (F12 → Console)

**Должно быть:**
```
Entry created successfully. Entries reloaded.
Loaded 1 entries for today
```

✅ Если есть → список перезагрузился!
❌ Если нет → список не обновился!

### Шаг 5: Проверьте Calendar

**В Calendar должно быть:**
```
Saved Today: ₫100,000
```

✅ Если показывает → Calendar правильно считает!
❌ Если показывает ₫0 → проблема в loadEntries!

### Шаг 6: Откройте Wallet

```
💰 Wallet → проверьте "Total Savings"
```

**Должно быть:**
```
Total Savings: ₫100,000
4 Points • Novice Saver
```

❌ Если показывает ₫0 → проблема в Wallet!

## 🐛 ВОЗМОЖНЫЕ ПРИЧИНЫ:

### Причина 1: База данных не обновляется

**Проверка:**
```powershell
npm run db:studio
```

Откроется Prisma Studio → Entry:
- Должна быть запись "Test Entry"
- usdAmount: 4.08
- currency: "VND"

❌ Если записи нет → проблема в API create!

### Причина 2: Wallet не перезагружается

**Проверка:**
```
1. После создания записи в Calendar
2. НЕ обновляйте страницу
3. Просто кликните на 💰 Wallet
```

✅ Если показывает ₫100,000 → всё ОК!
❌ Если показывает ₫0 → нужно обновить страницу (F5)

### Причина 3: Конвертация неправильная

**Проверка в Console (F12):**
```javascript
// Проверьте localStorage:
JSON.parse(localStorage.getItem('user'))

// Должно быть:
{ currency: "VND", language: "en", ... }
```

❌ Если currency: "USD" → смените снова!

## 🔧 ИСПРАВЛЕНИЕ (если нужно):

### Если Wallet не обновляется после создания записи:

**Решение 1: Обновите страницу после создания**
```
После создания записи → нажмите F5 в браузере
```

**Решение 2: Wallet должен перезагружать данные**

Проверим файл `app/wallet/page.tsx`:

```typescript
useEffect(() => {
  const parsedUser = getUserFromStorage();
  if (!parsedUser) {
    router.push('/');
    return;
  }
  setUser(parsedUser);
  setUserPoints(Number(parsedUser.points) || 0);
  loadStats(parsedUser.id);
}, [router]); // ← Загружается только при монтировании!
```

**Проблема:** Wallet загружается ТОЛЬКО при открытии, не после создания записи!

**Решение:** Добавить listener или обновлять при каждом фокусе.

### Вариант 2: Автоматическое обновление Wallet

Добавим код, чтобы Wallet обновлялся при каждом открытии:

```typescript
// app/wallet/page.tsx
useEffect(() => {
  // Перезагружать при каждом фокусе на странице
  const handleFocus = () => {
    if (user) {
      loadStats(user.id);
    }
  };
  
  window.addEventListener('focus', handleFocus);
  return () => window.removeEventListener('focus', handleFocus);
}, [user]);
```
