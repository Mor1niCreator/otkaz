## 🎯 ЛОГИКА ПРИЛОЖЕНИЯ:

### Savings (Сбережения) = Реальные деньги
```
User добавляет: Coffee $3
→ Entry создается: usdAmount = 3
→ Wallet: Total Savings = сумма всех usdAmount
→ Goals: Progress = totalSavings / goalTarget
```

### Points (Поинты) = Геймификация
```
User добавляет: Coffee $3 (category: drinks)
→ Base points: 3
→ Category bonus: 0 (только habits +20%)
→ Streak multiplier: 1.0 (увеличивается с днями подряд)
→ Total points: 3

User добавляет: Cigarettes $8 (category: habits)
→ Base points: 8
→ Category bonus: +20% = 9.6
→ Streak multiplier: 1.05 (5 дней подряд)
→ Total points: 10.08
```

## 🎮 ГЕЙМИФИКАЦИЯ (как работает):

### Начисление поинтов:
```javascript
// Базовая формула:
points = usdAmount * categoryBonus * streakMultiplier

// Примеры:
Coffee $3 (drinks, streak 0) = 3 * 1.0 * 1.0 = 3 points
Cigarettes $8 (habits, streak 0) = 8 * 1.2 * 1.0 = 9.6 points
Coffee $3 (drinks, streak 30) = 3 * 1.0 * 1.3 = 3.9 points
Cigarettes $8 (habits, streak 30) = 8 * 1.2 * 1.3 = 12.48 points
```

### Category Bonuses:
- **habits** (привычки): +20%
  - Cigarettes, Vaping, Alcohol
- **drinks, food, transport, entertainment, other**: 0%

### Streak Multipliers:
- 0-9 дней: 1.0x
- 10 дней: 1.1x
- 20 дней: 1.2x
- 30 дней: 1.3x
- ...
- 100+ дней: 2.0x (максимум)

### Ранги:
1. **Novice Saver** - 0+ points
2. **Habit Hacker** - 50+ points
3. **Frugal Master** - 150+ points
4. **Willpower Pro** - 300+ points
5. **Discipline Legend** - 500+ points

## 💰 SAVINGS vs POINTS:

### Savings (Сбережения):
- Отображаются в **Wallet** (Total Savings)
- Используются в **Goals** для прогресса
- Конвертируются в валюту пользователя
- **Всегда равны сумме всех записей**

### Points (Поинты):
- Отображаются в **Wallet** (рядом с рангом)
- Используются для **Achievements**
- Используются для **Ranks**
- **Могут быть больше чем savings** (из-за бонусов)

### Пример:
```
User добавил:
- Coffee $3 (drinks)
- Cigarettes $8 (habits)
- Soda $2 (drinks)

Savings = $3 + $8 + $2 = $13.00
Points = 3 + 9.6 + 2 = 14.6 points

Wallet показывает:
"Total Savings: $13.00"
"14.6 Points • Novice Saver"
```
