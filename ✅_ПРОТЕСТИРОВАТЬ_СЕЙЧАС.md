## 📋 ПЛАН ТЕСТИРОВАНИЯ (5 минут):

### ✅ ШАГ 1: Регистрация (ОБЯЗАТЕЛЬНО!)
```
1. Откройте: http://localhost:3000
2. Нажмите "Register"
3. Введите:
   Email: demo@test.com
   Password: demo123
   Name: Demo User
4. "Create Account"
5. ✅ Должно перебросить на Calendar
```

### ✅ ШАГ 2: Добавить запись
```
1. На странице Calendar
2. Нажмите ☕ Coffee (или "Кофе")
3. Форма откроется с ценой $3
4. Нажмите "Save" (или "Сохранить")
5. ✅ Должно: "+3.0 points! 🎉"
6. ✅ Запись появится в списке
7. ✅ "Saved Today: $3.00"
```

### ✅ ШАГ 3: Wallet с конвертацией
```
1. Нажмите 💰 Wallet внизу
2. ✅ Total Savings: $3.00
3. ✅ 3 Points • Novice Saver
4. ✅ График показывает данные
5. ✅ Today: $3.00
6. ✅ This Week: $3.00
```

### ✅ ШАГ 4: Goals с базовыми целями
```
1. Нажмите 🎯 Goals внизу
2. ✅ 4 цели автоматически созданы:
   📱 iPhone 17 Pro - $1,199.00
   🕶️ Ray-Ban Aviator Sunglasses - $165.00
   🎧 AirPods Pro - $249.00
   ✈️ Weekend Trip - $500.00
3. ✅ Прогресс-бары видны
4. ✅ iPhone: 0% ($3.00 / $1,199.00)
```

### ✅ ШАГ 5: Crypto-ROI
```
1. В Goals прокрутите вниз
2. Нажмите "🚀 Calculate Crypto ROI"
3. ✅ Показывает: "See what your $3.00 would be worth..."
4. ✅ Таблица с 10 криптовалютами
5. ✅ BTC: ~17.6x, ETH: ~21.3x
```

### ✅ ШАГ 6: Смена валюты → РУБЛИ
```
1. 👤 Profile
2. Нажмите "Edit"
3. Currency → выберите "RUB - Russian Ruble (₽)"
4. "Save"
5. Страница перезагрузится...
6. Вернитесь в 💰 Wallet
7. ✅ "Всего накоплено: ₽277.50"
8. Откройте 🎯 Goals
9. ✅ "iPhone 17 Pro - ₽110,925.00"
10. Откройте 📅 Calendar
11. ✅ "Сэкономлено сегодня: ₽277.50"
```

### ✅ ШАГ 7: Смена валюты → ДОНГИ
```
1. 👤 Profile → Edit
2. Currency → "VND - Vietnamese Dong (₫)"
3. Save
4. 💰 Wallet
5. ✅ "₫73,500" (БЕЗ копеек!)
6. 🎯 Goals
7. ✅ "iPhone 17 Pro - ₫29,375,500"
```

### ✅ ШАГ 8: Смена языка → РУССКИЙ
```
1. 👤 Profile → Edit
2. Language → "Русский 🇷🇺"
3. Save
4. Страница перезагрузится
5. ✅ Весь UI на русском:
   - "Календарь" (было Calendar)
   - "Кошелёк" (было Wallet)
   - "Цели" (было Goals)
   - "Достижения" (было Achievements)
   - "Профиль" (было Profile)
```

### ✅ ШАГ 9: Добавить свою цель
```
1. 🎯 Goals
2. Нажмите "+ Новая" (или "+ New")
3. Название: "MacBook Pro"
4. Сумма: 2500
5. "Сохранить"
6. ✅ Цель добавлена
7. ✅ Показана в вашей валюте
```

### ✅ ШАГ 10: Achievements
```
1. 🏅 Achievements (Достижения)
2. ✅ "☕ Coffee Breaker" разблокировано (добавили кофе)
3. ✅ Progress: 1/8 unlocked
4. ✅ Остальные 7 заблокированы
```

## 🐛 ЕСЛИ ЕСТЬ ОШИБКИ:

### "getUserFromStorage is not defined"
```powershell
# Перезапустите сервер:
Ctrl + C
npm run dev
```

### "Fast Refresh error"
```powershell
# Очистите кеш и перезапустите:
Ctrl + C
Remove-Item -Recurse -Force .next
npm run dev
```

### База данных пустая / нет целей
```powershell
# Пересоздайте базу:
Ctrl + C
Remove-Item prisma\dev.db
npm run db:init
npm run dev
```
