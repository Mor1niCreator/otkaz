## 🔄 ЧТО ДЕЛАТЬ СЕЙЧАС:

### ВАЖНО: Очистите старые данные и начните заново!

```powershell
# 1. Остановите сервер (если запущен)
Ctrl + C

# 2. Удалите старую базу данных
Remove-Item prisma\dev.db

# 3. Удалите кеш Next.js
Remove-Item -Recurse -Force .next

# 4. Пересоздайте базу
npm run db:init

# 5. Запустите сервер
npm run dev
```

### Затем в браузере:

1. **Зарегистрируйте НОВОГО пользователя:**
   - Email: `demo@test.com`
   - Password: `demo123`
   - Name: Demo User

2. **Добавьте запись:**
   - Calendar → ☕ Coffee → Save
   - Должно появиться: "+3.0 points! 🎉"

3. **Проверьте Wallet:**
   - Все суммы в USD
   - График отображается

4. **Проверьте Goals:**
   - Автоматически создались 4 цели:
     - 📱 iPhone 17 Pro - $1,199
     - 🕶️ Ray-Ban Aviator - $165
     - 🎧 AirPods Pro - $249
     - ✈️ Weekend Trip - $500

5. **Смените валюту:**
   - Profile → Edit → Currency → **RUB** (₽) → Save
   - Перезагрузится
   - ВСЕ суммы пересчитаются в рубли!

6. **Смените язык:**
   - Profile → Изменить → Язык → **Русский 🇷🇺** → Сохранить
   - Весь интерфейс на русском!
