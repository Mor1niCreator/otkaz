## ❌ ПРОБЛЕМА:

При смене на VND (вьетнамские донги) и создании записи:
- Сумма не прибавляется в кошельке
- Поинты начисляются неправильно

---

## 🐛 ЧТО БЫЛО НЕ ТАК:

### Старая логика (НЕПРАВИЛЬНАЯ):
```javascript
// lib/currency-service.ts (БЫЛО)
const rates = {
  EUR: 1.09,  // ❌ Это курс USD → EUR (для конвертации FROM USD)
  RUB: 0.011, // ❌ Это 1 RUB = 0.011 USD (неправильно!)
  VND: ???    // ❌ Вообще не было!
};

// Конвертация TO USD
return amount * rate; // ❌ НЕПРАВИЛЬНО!

// Пример:
// 24,500 VND * 0.000041 = 1 USD ❌ (но rate не был указан!)
```

### Проблема:
1. **Курсы были перепутаны** (FROM USD вместо TO USD)
2. **VND вообще не было в списке**
3. **Формула была неправильная** (умножение вместо деления)

## ✅ ИСПРАВЛЕНИЕ:

### Новая логика (ПРАВИЛЬНАЯ):
```javascript
// lib/currency-service.ts (СТАЛО)
const rates = {
  USD: 1,
  EUR: 0.92,    // ✅ 1 USD = 0.92 EUR
  RUB: 92.5,    // ✅ 1 USD = 92.5 RUB
  VND: 24500,   // ✅ 1 USD = 24,500 VND
  JPY: 149.5,   // ✅ 1 USD = 149.5 JPY
  // ... все 30+ валют
};

// Конвертация TO USD (деление)
const usdAmount = amount / rate; // ✅ ПРАВИЛЬНО!

// Примеры:
// 24,500 VND / 24,500 = 1 USD ✅
// 92.5 RUB / 92.5 = 1 USD ✅
// 0.92 EUR / 0.92 = 1 USD ✅
```

## 🔄 КАК ЭТО РАБОТАЕТ:

### Шаг 1: Пользователь меняет валюту
```
Profile → Edit → Currency → VND → Save
localStorage: user.currency = "VND"
```

### Шаг 2: Создание записи
```
Calendar → Custom Entry:
- Name: "Pho Soup"
- Price: 50,000 (VND)
- Quantity: 1

Frontend отправляет:
{
  name: "Pho Soup",
  pricePerUnit: 50000,
  quantity: 1,
  currency: "VND", // ← Берется из user.currency
  category: "food"
}
```

### Шаг 3: Backend конвертирует
```javascript
// pages/api/entries/create.ts
const totalAmount = 50000 * 1 = 50000 VND
const usdAmount = convertToUSD(50000, "VND")
                = 50000 / 24500
                = 2.04 USD ✅

// Сохраняет:
Entry {
  name: "Pho Soup",
  pricePerUnit: 50000,
  quantity: 1,
  currency: "VND",
  usdAmount: 2.04, // ← Правильно сконвертировано!
}
```

### Шаг 4: Начисление поинтов
```javascript
// lib/points-service.ts
const points = calculatePoints(2.04, "food", userId)
             = 2.04 * 1.0 * 1.0
             = 2.04 points ✅

User.points += 2.04
```

### Шаг 5: Отображение в Wallet
```javascript
// pages/api/goals/list.ts
const entries = [{usdAmount: 2.04}, ...]
const totalSavings = SUM(entries.usdAmount) = 2.04 USD

// Frontend (app/wallet/page.tsx)
const convertedSavings = convertCurrency(2.04, "VND")
                       = 2.04 * 24500
                       = 49,980 ≈ ₫50,000 ✅

Display: "Tổng tiết kiệm: ₫49,980"
```
