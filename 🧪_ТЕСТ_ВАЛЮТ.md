## 📊 МАТЕМАТИКА (как это работает):

### Конвертация TO USD:
```javascript
// 1 USD = 24,500 VND
// Конвертация VND → USD: делим на курс

50,000 VND / 24,500 = 2.04 USD ✅
100,000 VND / 24,500 = 4.08 USD ✅
25,000 VND / 24,500 = 1.02 USD ✅
```

### Конвертация FROM USD:
```javascript
// 1 USD = 24,500 VND
// Конвертация USD → VND: умножаем на курс

2.04 USD × 24,500 = 49,980 ≈ 50,000 VND ✅
4.08 USD × 24,500 = 99,960 ≈ 100,000 VND ✅
```

### Поинты:
```javascript
// Points = usdAmount × bonuses

50,000 VND = 2.04 USD
→ Points: 2.04 × 1.0 × 1.0 = 2.04 ✅

(если habits: 2.04 × 1.2 = 2.45)
(если streak 30: 2.04 × 1.3 = 2.65)
```

## 🔍 ОТЛАДКА:

### Если сумма не прибавляется:

#### 1. Проверьте логи в PowerShell:
```
Entry: XXX - AMOUNT CURRENCY = X.XX USD
Points earned: X.XX (from X.XX USD)
POST /api/entries/create 201 in XXXms
```

Если нет этих строк → запись не создалась!

#### 2. Проверьте Network в браузере:
```
F12 → Network → POST /api/entries/create

Request:
{
  "name": "Pho Soup",
  "pricePerUnit": 50000,
  "quantity": 1,
  "currency": "VND", ← Должен быть VND!
  "category": "food"
}

Response (201):
{
  "entry": {...},
  "pointsEarned": 2.04
}
```

#### 3. Проверьте базу данных:
```powershell
npm run db:studio
```

Откроется Prisma Studio → Entry:
- usdAmount должен быть ~2.04 (не 50000!)
- currency должен быть "VND"

#### 4. Проверьте localStorage:
```javascript
// В консоли браузера (F12):
JSON.parse(localStorage.getItem('user'))

// Должно быть:
{
  "currency": "VND", ← Правильная валюта
  "language": "en" или "ru",
  "points": 2.04
}
```

## ❌ ТИПИЧНЫЕ ОШИБКИ:

### Ошибка 1: Старый кеш
```
Симптом: Сумма не обновляется
Решение: Remove-Item -Recurse -Force .next
```

### Ошибка 2: Валюта не сохранилась
```
Симптом: Создается запись в USD вместо VND
Решение: 
1. Profile → Edit → VND → Save
2. Обязательно дождитесь reload страницы
3. Проверьте localStorage (F12 → Console)
```

### Ошибка 3: Запись создается, но не видна
```
Симптом: Toast "+X points", но записи нет
Решение: 
1. Обновите страницу (F5)
2. Проверьте loadEntries() работает
```

### Ошибка 4: Неправильные поинты
```
Симптом: 50,000 VND → 50,000 points
Проблема: usdAmount = 50,000 вместо 2.04
Решение: Убедитесь что currency правильный!
```
