## 🎨 Стиль и рекомендации

### Общий стиль
- **Комикс-стиль**: Простые линии, яркие цвета, минимализм
- **Цветовая палитра**: 
  - Primary: #4ade80 (зеленый)
  - Secondary: #f59e0b (оранжевый)
  - Accent: #f97316 (красно-оранжевый)
- **Формат**: SVG для векторной графики, PNG для PWA иконок
- **Размеры**: Указаны в пикселях для каждого файла

### Технические требования
- **SVG**: Должны быть оптимизированы, без лишних элементов
- **PNG**: Высокое качество, прозрачный фон
- **Совместимость**: Работать на темном и светлом фоне
- **Доступность**: Контрастные цвета, читаемость

### Где используются
- **Навигация**: Логотип в шапке
- **Формы**: Иконки пресетов в EntryForm
- **Цели**: Иконки в Goals и GoalCard
- **Достижения**: Бейджи в Achievements
- **Ранги**: Значки в Ranks
- **PWA**: Иконки для установки приложения

## 🔄 Процесс замены

1. **Подготовка**: Создайте файлы в указанных размерах
2. **Замена**: Скопируйте файлы в соответствующие папки
3. **Тестирование**: Проверьте отображение на всех экранах
4. **Оптимизация**: Сожмите файлы для веба
5. **PWA**: Обновите манифест при необходимости

## 📝 Примечания

- Все файлы должны быть в формате SVG (кроме PWA иконок)
- Используйте единый стиль для всех иконок
- Проверьте отображение на мобильных устройствах
- Убедитесь в соответствии цветовой схеме приложения
- Тестируйте на разных разрешениях экрана
