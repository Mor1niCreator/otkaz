### 1. ✅ Цели теперь показываются

**Файл:** `pages/api/goals/list.ts`

**Проблема:**
```typescript
// БЫЛО (НЕПРАВИЛЬНО):
const user = await prisma.user.findUnique({
  where: { id: userId as string },
  select: { points: true },
});
const totalSavings = user?.points || 0; // ❌ Points ≠ Savings
```

**Исправление:**
```typescript
// СТАЛО (ПРАВИЛЬНО):
const entries = await prisma.entry.findMany({
  where: { userId: userId as string },
  select: { usdAmount: true },
});
const totalSavings = entries.reduce((sum, entry) => sum + entry.usdAmount, 0);
// ✅ Savings = сумма всех записей
```

**Результат:**
- Goals теперь показывают правильный прогресс
- Total Savings считается из реальных записей
- 4 базовые цели создаются автоматически

      setShowCrypto(true);

    setLoadingCrypto(false);
  }
};

### 4. ✅ Система поинтов теперь понятная

**Концепция исправлена:**

#### Savings (Сбережения) = Реальные деньги
```
Формула: SUM(всех entries.usdAmount)

Где используется:
- Wallet: "Total Savings: $XX.XX"
- Goals: прогресс = totalSavings / goalTarget
- Crypto-ROI: yourValue = totalSavings * multiplier

Пример:
User добавил:
- Coffee $3
- Taxi $15
- Soda $2
→ Total Savings = $20
```

#### Points (Поинты) = Геймификация
```
Формула: usdAmount * categoryBonus * streakMultiplier

Category Bonuses:
- habits: 1.2x (+20%)
- остальное: 1.0x

Streak Multiplier:
- 0-9 дней: 1.0x
- 10 дней: 1.1x
- 30 дней: 1.3x
- 100+ дней: 2.0x (max)

Где используется:
- Wallet: "X Points • Rank"
- Achievements: unlock conditions
- Ranks: level progression

Пример:
User добавил:
- Coffee $3 (drinks, streak 0) → 3 * 1.0 * 1.0 = 3 points
- Cigarettes $8 (habits, streak 10) → 8 * 1.2 * 1.1 = 10.56 points
→ Total Points = 13.56
→ Total Savings = $11
```

**Wallet UI:**
```
┌─────────────────────────────────┐
│ Total Savings: $11.00           │ ← Savings
│ 13.56 Points • Novice Saver     │ ← Points + Rank
└─────────────────────────────────┘
```
